minimal-site-block

A Firefox add-on for blocking sites easily.

Introduction

This Add-on provides you with a (very) basic interface to block websites.

What does this plugin do?

Minimal Site Block has been developed to block sites. You can specify the sites in the add-on settings.

At the moment the following block operations are supported:

  • Block specific URL (with Backslash!): http://www.firefox.com/
  • Block URL and all pages under this URL: http://www.firefox.com/*
  • Block all URLs containing the pattern (with one wildcard (*) at maximum): *.firefox.com
  • Block files or data: file://*, data:*

You can change the settings in the Extensions (about:addons).

What does this plugin not do?

Minimal Site Block will not clutter your Firefox with icons, toolbars or buttons.

Current limitations

  • There can only be one wildcard at a time due using the wildcard option.
